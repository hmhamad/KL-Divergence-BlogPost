# Forward vs Reverse KL Divergence

Companion code for the blog post [Forward vs Reverse KL Divergence: Why the Direction Matters](https://www.hassanhamad.com/blog/forward-vs-reverse-kl/forward-vs-reverse-kl.html).

## Code

All experiment code is in `main.py`.

## Setup & Run

Using [uv](https://docs.astral.sh/uv/) (or any Python package manager of your choice):

```bash
uv sync
uv run main.py
```
